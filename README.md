
# 🌐 WebStudio Page — Modern Business Website

Un website de prezentare elegant și responsiv, construit pentru a demonstra abilități solide de front‑end development. Proiectul pune accent pe design modern, structură clară și o experiență vizuală potrivită pentru companii, agenții sau freelanceri.

🔗 **Demo Live:** https://anisoara1.github.io/WebStudio-page/

---

## ✨ Caracteristici principale

- **Design modern și profesional**, potrivit pentru branduri digitale.
- **Navigație intuitivă** cu meniu fix și secțiuni clare.
- **Responsive design** pentru desktop, tabletă și mobil.
- **Secțiuni dedicate** pentru servicii, portofoliu, echipă și contact.
- **Efecte vizuale** (hover, animații subtile, layout fluid).
- **Cod organizat**, ușor de extins și întreținut.

---

## 🛠️ Tehnologii utilizate

| Tehnologie | Utilizare |
